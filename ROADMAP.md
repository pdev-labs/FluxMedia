# FluxMedia Project Roadmap

This document outlines the milestones and future plans for FluxMedia development.

## 🚀 Future Milestones

### Milestone 1: Core Downloader Enhancements (Target: Q3 2026)
- [ ] **Multi-threaded Downloads**: Implement chunked parallel down-loaders for raw HTTP links to maximize bandwidth.
- [ ] **Expanded Extractor Presets**: Add first-class support for Bilibili, SoundCloud, and Vimeo custom properties.
- [ ] **Advanced Subtitle Parsing**: Support styling translation injection for subtitles.

---

### Milestone 2: TUI/CLI Experience (Target: Q4 2026)
- [ ] **Interactive Visual Trimmer**: Provide an in-terminal timeline selector for cutting video/audio segments before downloading.
- [ ] **Custom Style Themes Engine**: Allow styling of TUI colors dynamically via standard CSS or TOML configuration files.
- [ ] **Download Queue Manager**: Implement pause/resume and prioritization within a persistent queue.

---

### Milestone 3: Share Portal (Target: Q1 2027)
- [ ] **Multi-user Authentication**: Define guest profiles vs admin users with custom permissions.
- [ ] **Search & Tags Filter**: Improve browser UI to support search, sort, and custom tags filtering.
- [ ] **Web Video Player Upgrades**: Integrate HLS/Dash streaming support and customized subtitle overlays on the web client player.
