Dynamic Console TUI

Experience a next-generation terminal user interface. FluxMedia renders rich interactive tables, beautifully structured configuration panels, and live progress bars showing real-time download speeds, ETA, and exact file sizes directly in your command prompt. It's fully responsive to your terminal window size!

🎵

Smart Audio Extraction

Looking to build your local music library? FluxMedia can automatically download video files, extract the audio tracks seamlessly, and convert them into high-quality MP3s (up to 192kbps). Even better, it embeds the YouTube video thumbnail as the album cover art and writes all song metadata directly into the file!

🌐

LAN QR Share Portal

Stop transferring files via USB. With our built-in LAN sharing tool, you can instantly host a beautiful web interface on your local network. It generates a scannable QR code right in your terminal, allowing any phone, tablet, or smart TV on your Wi-Fi to browse, stream, and download your entire media collection instantly.

📚

Batch Playlist Downloads

Save massive amounts of content with a single command. FluxMedia supports fetching entire playlists, recent channel uploads, or custom URL lists. It intelligently skips already downloaded files, tracks failed downloads, and organizes everything into neatly named directories so you never lose track of your series.

🏷️

Rich Metadata Embedding

Keep your files perfectly organized for local media servers like Plex, Jellyfin, or Kodi. FluxMedia automatically captures video descriptions, original upload dates, channel names, and high-resolution thumbnails, and securely embeds them into the `.mp4` or `.mkv` file properties natively.

⚙️

Deep Global Settings

Make FluxMedia truly yours. The built-in configuration manager allows you to set default download folders, establish strict format preferences (e.g., "always download 1080p MP4"), define custom filename structures to include upload dates, and switch between multiple premium terminal color themes.

Advanced CLI Capabilities

For the Power Users

While the TUI is great for everyday use, FluxMedia was built from the ground up as a robust CLI application. By passing arguments directly to the program, you can script, automate, and trigger complex download sequences silently in the background.

  • Subtitles Capturing: Download caption files directly for specified language codes (e.g., `en`, `es`) and automatically embed them as soft-subs into the video container.
  • Cookie Authentication: Pass `cookies.txt` files to authenticate your requests, allowing you to securely download age-restricted, private, or members-only videos from your own account.
  • Hardware Acceleration: Automatically leverages FFmpeg hardware acceleration on supported systems to drastically speed up audio extraction and video merging processes without bottlenecking your CPU.
  • Rate Limiting & Proxies: Set maximum download speed limits to prevent network hogging, and route your download traffic through HTTP/SOCKS5 proxies to bypass regional restrictions.

Quick Installation

Install FluxMedia globally on any system running Python 3.10+ directly via pip:

pip install fluxmedia

Requires FFmpeg for merging high-definition video formats and converting audio to MP3.
