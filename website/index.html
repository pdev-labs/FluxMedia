🌊 Welcome to FluxMedia TUI...

Everything you need in one tool

🎨

Dynamic Console TUI

Renders rich tables, panels, and live progress bars for download speeds, ETA, and file sizes directly in your terminal.

🎵

Smart Audio Extraction

Automatically extract and convert media into high-quality MP3 (192kbps) with embedded cover art and metadata.

🌐

LAN QR Share Portal

Easily share downloaded media over your local network using an interactive web portal generated on the fly.

📚

Batch Downloads

Save entire playlists or recent uploads from content creator channels with just a few keystrokes.

🏷️

Metadata Embedding

Automatically write video thumbnails and details (description, uploader, dates) directly into your downloads.

⚙️

Global Settings

Customize download folders, default formats, filename structures, and UI themes inside the app.

Quick Installation

Install FluxMedia globally on any system running Python 3.10+ directly via pip:

pip install fluxmedia

Requires FFmpeg for merging high-definition video formats and converting audio to MP3.
