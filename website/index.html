Quick Installation

Run the automated one-liner in your terminal to safely install Python, FFmpeg, and FluxMedia in seconds.

macOS & Linux (Terminal)

curl -sL https://raw.githubusercontent.com/pdev-labs/FluxMedia/main/install.sh | bash

Windows (PowerShell)

irm https://raw.githubusercontent.com/pdev-labs/FluxMedia/main/install.ps1 | iex
